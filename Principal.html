Diferentes maneras de agregar código JavaScript

Cualquier script de JavaScript, no importa cómo lo utilices, puede tener 2 tipos de "código"

Tipo 1: Las funciones o métodos que tú declares.

Estos siempre se ejecutarán solamente cuando tú los invoques desde cualquier parte del programa.

Tipo 2: El código o las instrucciones que NO forman parte de una función o método.

Este código puede considerarse como "el programa princial" de tu script.

Se ejecutará automáticamente al momento de carguar la página o documento que lo contiene.

Este código siempre se ejecuta en el órden en que es cargado en el navegador

Por ello resulta importante el órden en que incrustas o cargas los scripts en el documento.

El navegador ejecuta cada script que incluyas en el documento, en el órden que van apareciendo.

pero se ejecutan como un único programa.

Al ejecutar un script en particular, el navegador conserva el "estado de la ejecución" de los script anteriores,

pero desconoce completamente la existencia de los scripts posteriores.
